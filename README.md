Diabetic Retinopathy Detection

📌 Project Description

This project is developed using MATLAB GUI for detecting diabetic retinopathy from retinal images.

The system:

- Loads retinal images
- Extracts features like mean, entropy, skewness, and kurtosis
- Displays results using GUI
- Evaluates performance using accuracy, sensitivity, and specificity

⚙️ How to Run

1. Open MATLAB
2. Run "FG.m"
3. Upload an image
4. View results in GUI

🧠 Technology Used

- MATLAB
- Image Processing
- GUI (GUIDE)

📄 Files Included

- FG.m (Main file)
- final_report.pdf (Project report)
